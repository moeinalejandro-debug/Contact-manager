#  Contact Manager

A simple and user-friendly Contact Manager application built with Python. It allows users to store, search, edit, and remove contacts using a JSON file for persistent storage.

##  Features

*  Add new contacts
*  View all contacts
*  Search contacts by name or phone number
*  Edit contact name or phone number
*  Remove contacts
*  Automatic data saving using JSON
*  Persistent storage between program runs

##  Technologies

* Python 3
* JSON

## Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/contact-manager.git
```

Go to the project directory:

```bash
cd contact-manager
```

Run the application:

```bash
python main.py
```
